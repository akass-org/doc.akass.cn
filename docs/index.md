# 欢迎

这里是 AKASS.CN 的主页。以下是提供的服务。

> 注意：所有的服务均面向中国大陆，境外访问并不是我们的首要考虑。
> 如果您的用户以境外用户为主，还是推荐您使用原服务而非 `AKASS`。

> 除图床外，其他服务均只向好友提供服务。实在是没钱撑起这些东西。
